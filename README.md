## Installing on Ubuntu and its Derivatives

```sh
wget -q --show-progress https://github.com/er-knight/contests/releases/download/v0.1/contests.zip
```
```sh
unzip contests.zip
```
```sh
echo "export PATH=\$PATH:~/contests" >> .bashrc
```
```sh
source .bashrc
```
